How to use:
Export to addons.

To make yourself "Owner" (It's the exact same as Superadmin, except different color and name in the scoreboard)
	Go to addons/Divrans Admin Mod/lua/autorun/d_teams.lua
	Put your Steam ID in the lua file.

To make someone admin or super admin:
	Listen server:
	Go to steam/steamapps/yourname/garrysmod/garrysmod/settings/users.txt
	Put your name and steamID in it.

	Dedicated server:
	Go to server folder/garrysmod/garrysmod/settings/users.txt
	Put your name and steamID in it.
